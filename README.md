# Time Series Visualizer

This project is part of the **freeCodeCamp Data Analysis with Python certification**.

It visualizes time series data of daily page views on the freeCodeCamp forum using **Pandas, Matplotlib, and Seaborn**.

## Features

The project generates three visualizations:

- **Line Plot** – Shows daily forum page views over time.
- **Bar Plot** – Shows the average daily page views for each month grouped by year.
- **Box Plots**
  - Year-wise Box Plot (Trend)
  - Month-wise Box Plot (Seasonality)

## Data Cleaning

Before visualization, the dataset is cleaned by removing the top 2.5% and bottom 2.5% of page view values to eliminate outliers.
